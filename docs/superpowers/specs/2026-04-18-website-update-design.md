# Design Spec: MalikClaw Website Update (Global Gryphon Intelligence)

**Date:** April 18, 2026
**Topic:** Website Modernization for Global AI Infrastructure
**Status:** Draft

---

## 1. Vision & Strategy

### Objective
Transform the MalikClaw website into a world-class, professional showcase for a production-grade autonomous agent framework. The site must convey **Intelligence**, **Efficiency**, and **Global Reliability**.

### Target Audience
- **Developers:** Looking for lightweight agent infrastructure.
- **Enterprise Architects:** Seeking secure, privacy-first AI solutions.
- **Edge Engineers:** Needing AI that runs on $10 hardware.

### Key Vibe
**Powerful & Intelligent.** A high-end, "Midnight Onyx" aesthetic with gold accents (Gryphon Branding) and technical transparency (showing the Agent Loop).

---

## 2. Visual Design (The "Gryphon" Identity)

### Color Palette
- **Background:** `#020202` (Midnight Onyx)
- **Primary Accent:** `#EAB308` (Gryphon Gold)
- **Secondary Accent:** `#3B82F6` (Electric Blue - for technical/data elements)
- **Text:** `#F4F4F5` (Zinc 100) / `#A1A1AA` (Zinc 400)

### Typography
- **Primary:** `Geist Sans` or `Inter` (Ultra-modern, clean, global)
- **Monospace:** `Geist Mono` or `JetBrains Mono` (For technical logs and CLI)
- **Urdu:** (Removed decorative headings; restricted to functional translation toggle)

---

## 3. Architecture & Components

### A. The Hero Section (The "Neural Loop")
- **Headline:** "MalikClaw: The Autonomous Agent Infrastructure for Every Device."
- **Visual Hero:** An animated SVG/Framer-Motion diagram of the **PLAN → ACT → OBSERVE → REFLECT → MEMORY UPDATE** cycle.
- **Interaction:** A "thought pulse" travels through nodes. A translucent side-terminal shows live logs for a sample task (e.g., "Take a screenshot of my phone").
- **CTA:** "Launch MalikClaw" (Gold Gradient) & "Explore Architecture" (Ghost Button).

### B. The "Core Power" Feature Grid
- **Efficiency Card:** Highlights <10MB RAM, <1s Boot, $10 Hardware.
- **Agentic Card:** Highlights the production-grade loop and autonomous execution.
- **Toolbox Card:** Highlights ADB, Gmail, Web Search, and MCP support.
- **Privacy Card:** Highlights 100% local-first, secure-by-default architecture.

### C. Performance Comparison Table
- Glassmorphism table comparing MalikClaw to standard frameworks (LangChain, AutoGPT, etc.) on footprint and speed.

### D. Global Installation Section
- One-command installers prominently featured with a "Copy to Clipboard" interaction.

---

## 4. Technical Implementation

- **Framework:** Next.js 15+ (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion (for the Neural Loop and smooth transitions)
- **Icons:** Lucide React
- **Responsiveness:** Mobile-first, ensuring the SVG diagram scales elegantly.

---

## 5. Success Criteria
1. The site feels "expensive" and professional upon landing.
2. The complex "Agent Loop" is understood within 5 seconds of viewing.
3. The ultra-lightweight nature is clearly proven by data.
4. The site is fully functional and responsive on all devices.
