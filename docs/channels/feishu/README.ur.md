# فی شو (Feishu)

فی شو (بین الاقوامی ورژن کا نام: Lark) بائٹ ڈانس (ByteDance) کے تحت ایک انٹرپرائز تعاون کا پلیٹ فارم ہے۔ یہ ایونٹ سے چلنے والے ویب ہک (Webhook) کے ذریعے چین اور عالمی مارکیٹوں کو سپورٹ کرتا ہے۔

## کنفیگریشن

```json
{
  "channels": {
    "feishu": {
      "enabled": true,
      "app_id": "cli_xxx",
      "app_secret": "xxx",
      "encrypt_key": "",
      "verification_token": "",
      "allow_from": []
    }
  }
}
```

| فیلڈ | قسم | لازمی | تفصیل |
| ------------------ | ------ | ---- | -------------------------------- |
| enabled | bool | ہاں | کیا فی شو چینل کو فعال کرنا ہے |
| app_id | string | ہاں | فی شو ایپ کی ایپ آئی ڈی (cli_ سے شروع ہوتی ہے) |
| app_secret | string | ہاں | فی شو ایپ کا ایپ سیکریٹ (App Secret) |
| encrypt_key | string | نہیں | ایونٹ کال بیک کی انکرپشن کی (Encryption Key) |
| verification_token | string | نہیں | ویب ہک ایونٹ کی تصدیق کے لیے ٹوکن |
| allow_from | array | نہیں | یوزر آئی ڈی کی وائٹ لسٹ، خالی کا مطلب ہے تمام صارفین |
| random_reaction_emoji | array | نہیں | تصادفی طور پر شامل کیے گئے ایموجیز کی فہرست، خالی ہونے پر "Pin" استعمال ہوگا |

## ترتیب کا عمل

1. ایپلیکیشن بنانے کے لیے [فی شو اوپن پلیٹ فارم](https://open.feishu.cn/) پر جائیں
2. ایپ آئی ڈی اور ایپ سیکریٹ حاصل کریں
3. ایونٹ سبسکرپشن اور ویب ہک URL ترتیب دیں
4. انکرپشن سیٹ کریں (اختیاری، پروڈکشن ماحول کے لیے تجویز کردہ)
5. ایپ آئی ڈی، ایپ سیکریٹ، انکرپٹ کی (Encrypt Key) اور ویریفیکیشن ٹوکن (اگر انکرپشن فعال ہے) کو کنفیگریشن فائل میں درج کریں
6. ان ایموجیز کو حسب ضرورت بنائیں جن کے ساتھ آپ چاہتے ہیں کہ MalikClaw آپ کے پیغامات پر ردعمل دے (اختیاری، حوالہ کے لیے: [Feishu Emoji List](https://open.larkoffice.com/document/server-docs/im-v1/message-reaction/emojis-introduce))
