# MaixCam

میکس کیم (MaixCam) ایک ایسا چینل ہے جو خاص طور پر Sipeed MaixCAM اور MaixCAM2 AI کیمرہ آلات کو جوڑنے کے لیے ڈیزائن کیا گیا ہے۔ یہ دو طرفہ مواصلت کے لیے TCP ساکٹ کا استعمال کرتا ہے اور ایج AI (Edge AI) کی تعیناتی کے منظرناموں کو سپورٹ کرتا ہے۔

## کنفیگریشن

```json
{
  "channels": {
    "maixcam": {
      "enabled": true,
      "server_address": "0.0.0.0:8899",
      "allow_from": []
    }
  }
}
```

| فیلڈ | قسم | لازمی | تفصیل |
| -------------- | ------ | ---- | -------------------------------- |
| enabled | bool | ہاں | کیا میکس کیم چینل کو فعال کرنا ہے |
| server_address | string | ہاں | TCP سرور کے سننے کا پتہ اور پورٹ |
| allow_from | array | نہیں | ڈیوائس آئی ڈی کی وائٹ لسٹ، خالی کا مطلب ہے تمام آلات کی اجازت ہے |

## استعمال کے منظرنامے

میکس کیم چینل MalikClaw کو ایج ڈیوائسز کے AI بیک اینڈ کے طور پر کام کرنے کے قابل بناتا ہے:

- **ذہین نگرانی** : میکس کیم امیج فریم بھیجتا ہے، اور MalikClaw وژن ماڈلز کے ذریعے ان کا تجزیہ کرتا ہے
- **آئی او ٹی (IoT) کنٹرول** : آلات سینسر کا ڈیٹا بھیجتے ہیں، اور MalikClaw ردعمل کو مربوط کرتا ہے
- **آف لائن AI** : کم لیٹنسی (latency) کے لیے مقامی نیٹ ورک پر MalikClaw کو تعینات کریں
