# ڈنگ ٹاک (DingTalk)

ڈنگ ٹاک علی بابا کا انٹرپرائز کمیونیکیشن پلیٹ فارم ہے، جو چین میں بہت مقبول ہے۔ یہ مستقل کنکشن برقرار رکھنے کے لیے سٹریمنگ SDK استعمال کرتا ہے۔

## کنفیگریشن

```json
{
  "channels": {
    "dingtalk": {
      "enabled": true,
      "client_id": "YOUR_CLIENT_ID",
      "client_secret": "YOUR_CLIENT_SECRET",
      "allow_from": []
    }
  }
}
```

| فیلڈ | قسم | لازمی | تفصیل |
| ------------- | ------ | ---- | -------------------------------- |
| enabled | bool | ہاں | کیا ڈنگ ٹاک چینل کو فعال کرنا ہے |
| client_id | string | ہاں | ڈنگ ٹاک ایپ کی کلائنٹ آئی ڈی (Client ID) |
| client_secret | string | ہاں | ڈنگ ٹاک ایپ کا کلائنٹ سیکریٹ (Client Secret) |
| allow_from | array | نہیں | یوزر آئی ڈی کی وائٹ لسٹ، خالی کا مطلب ہے تمام صارفین کی اجازت ہے |

## ترتیب کا عمل

1. [ڈنگ ٹاک اوپن پلیٹ فارم](https://open.dingtalk.com/) پر جائیں
2. ایک اندرونی انٹرپرائز ایپ بنائیں
3. ایپ سیٹنگز سے کلائنٹ آئی ڈی اور کلائنٹ سیکریٹ حاصل کریں
4. OAuth اور ایونٹ سبسکرپشن کنفیگر کریں (اگر ضرورت ہو)
5. کلائنٹ آئی ڈی اور کلائنٹ سیکریٹ کو کنفیگریشن فائل میں درج کریں
