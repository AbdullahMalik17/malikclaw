# Slack

سلیک (Slack) دنیا کا صف اول کا انٹرپرائز لیول فوری پیغام رسانی (instant messaging) کا پلیٹ فارم ہے۔ MalikClaw ریئل ٹائم دو طرفہ مواصلت حاصل کرنے کے لیے سلیک کے ساکٹ موڈ (Socket Mode) کا استعمال کرتا ہے، جس کے لیے عوامی ویب ہک (Webhook) اینڈ پوائنٹ ترتیب دینے کی ضرورت نہیں ہوتی۔

## کنفیگریشن

```json
{
  "channels": {
    "slack": {
      "enabled": true,
      "bot_token": "xoxb-...",
      "app_token": "xapp-...",
      "allow_from": []
    }
  }
}
```

| فیلڈ | قسم | لازمی | تفصیل |
| ---------- | ------ | ---- | -------------------------------------------------------- |
| enabled | bool | ہاں | کیا سلیک چینل کو فعال کرنا ہے |
| bot_token | string | ہاں | سلیک بوٹ کا بوٹ یوزر OAuth ٹوکن (xoxb- سے شروع ہوتا ہے) |
| app_token | string | ہاں | سلیک ایپ کا ساکٹ موڈ ایپ لیول ٹوکن (xapp- سے شروع ہوتا ہے) |
| allow_from | array | نہیں | یوزر آئی ڈی کی وائٹ لسٹ، خالی کا مطلب ہے تمام صارفین کی اجازت ہے |

## ترتیب کا عمل

1. ایک نئی سلیک ایپ بنانے کے لیے [Slack API](https://api.slack.com/) پر جائیں
2. ساکٹ موڈ کو فعال کریں اور ایپ لیول ٹوکن حاصل کریں
3. بوٹ ٹوکن اسکوپس (مثلاً `chat:write` ، `im:history` وغیرہ) شامل کریں
4. ورک اسپیس میں ایپ انسٹال کریں اور بوٹ یوزر OAuth ٹوکن حاصل کریں
5. بوٹ ٹوکن اور ایپ ٹوکن کو کنفیگریشن فائل میں درج کریں
