# Telegram

ٹیلی گرام (Telegram) چینل ٹیلی گرام بوٹ API کا استعمال کرتے ہوئے لانگ پولنگ (long polling) کے ذریعے مواصلت کرتا ہے۔ یہ ٹیکسٹ پیغامات، میڈیا فائلز (تصاویر، آواز، آڈیو، دستاویزات)، Groq Whisper کے ذریعے آواز کی ٹرانسکرپشن، اور بلٹ ان کمانڈ پروسیسر کو سپورٹ کرتا ہے۔

## کنفیگریشن

```json
{
  "channels": {
    "telegram": {
      "enabled": true,
      "token": "123456789:ABCdefGHIjklMNOpqrsTUVwxyz",
      "allow_from": ["123456789"],
      "proxy": ""
    }
  }
}
```

| فیلڈ | قسم | لازمی | تفصیل |
| ---------- | ------ | ---- | --------------------------------------------------------- |
| enabled | bool | ہاں | کیا ٹیلی گرام چینل کو فعال کرنا ہے |
| token | string | ہاں | ٹیلی گرام بوٹ API ٹوکن |
| allow_from | array | نہیں | یوزر آئی ڈی کی وائٹ لسٹ، خالی کا مطلب ہے تمام صارفین کی اجازت ہے |
| proxy | string | نہیں | ٹیلی گرام API سے جڑنے کے لیے پراکسی URL (مثلاً http://127.0.0.1:7890) |

## ترتیب کا عمل

1. ٹیلی گرام میں `@BotFather` تلاش کریں
2. `/newbot` کمانڈ بھیجیں اور نیا بوٹ بنانے کے لیے ہدایات پر عمل کریں
3. HTTP API ٹوکن حاصل کریں
4. ٹوکن کو کنفیگریشن فائل میں درج کریں
5. (اختیاری) ان صارفین کو محدود کرنے کے لیے `allow_from` ترتیب دیں جنہیں بات چیت کی اجازت ہے (آئی ڈی حاصل کرنے کے لیے `@userinfobot` استعمال کیا جا سکتا ہے)
