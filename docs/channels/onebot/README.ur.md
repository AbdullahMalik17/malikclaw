# OneBot

ون بوٹ (OneBot) کیو کیو (QQ) بوٹس کے لیے ایک اوپن پروٹوکول سٹینڈرڈ ہے، جو مختلف کیو کیو بوٹ نفاذ (مثلاً go-cqhttp، Mirai) کے لیے ایک یکساں انٹرفیس فراہم کرتا ہے۔ یہ مواصلت کے لیے WebSocket کا استعمال کرتا ہے۔

## کنفیگریشن

```json
{
  "channels": {
    "onebot": {
      "enabled": true,
      "ws_url": "ws://localhost:8080",
      "access_token": "",
      "allow_from": []
    }
  }
}
```

| فیلڈ | قسم | لازمی | تفصیل |
| ------------ | ------ | ---- | -------------------------------- |
| enabled | bool | ہاں | کیا OneBot چینل کو فعال کرنا ہے |
| ws_url | string | ہاں | OneBot سرور کا WebSocket URL |
| access_token | string | نہیں | OneBot سرور سے جڑنے کے لیے ایکسیس ٹوکن |
| allow_from | array | नहीं | یوزر آئی ڈی کی وائٹ لسٹ، خالی کا مطلب ہے تمام صارفین کی اجازت ہے |

## ترتیب کا عمل

1. ایک OneBot سے مطابقت رکھنے والا نفاذ (مثلاً napcat) تعینات کریں
2. WebSocket سروس کو فعال کرنے اور ایکسیس ٹوکن (اگر ضرورت ہو) سیٹ کرنے کے لیے OneBot نفاذ کو کنفیگر کریں
3. WebSocket URL اور ایکسیس ٹوکن کو کنفیگریشن فائل میں درج کریں
