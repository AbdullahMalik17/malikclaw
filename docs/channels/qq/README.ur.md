# QQ

MalikClaw کیو کیو (QQ) اوپن پلیٹ فارم کے آفیشل بوٹ API کے ذریعے کیو کیو کے لیے سپورٹ فراہم کرتا ہے۔

## کنفیگریشن

```json
{
  "channels": {
    "qq": {
      "enabled": true,
      "app_id": "YOUR_APP_ID",
      "app_secret": "YOUR_APP_SECRET",
      "allow_from": []
    }
  }
}
```

| فیلڈ | قسم | لازمی | تفصیل |
| ---------- | ------ | ---- | -------------------------------- |
| enabled | bool | ہاں | کیا کیو کیو چینل کو فعال کرنا ہے |
| app_id | string | ہاں | کیو کیو بوٹ ایپ کی ایپ آئی ڈی (App ID) |
| app_secret | string | ہاں | کیو کیو بوٹ ایپ کا ایپ سیکریٹ (App Secret) |
| allow_from | array | نہیں | یوزر آئی ڈی کی وائٹ لسٹ، خالی کا مطلب ہے تمام صارفین کی اجازت ہے |

## ترتیب کا عمل

1. بوٹ بنانے کے لیے [QQ اوپن پلیٹ فارم](https://q.qq.com/) پر جائیں
2. ڈیش بورڈ کے ذریعے ایپ آئی ڈی اور ایپ سیکریٹ حاصل کریں
3. بوٹ سینڈ باکس موڈ کو آن کریں، اور صارفین اور گروپس کو سینڈ باکس میں شامل کریں
4. ایپ آئی ڈی اور ایپ سیکریٹ کو کنفیگریشن فائل میں درج کریں
