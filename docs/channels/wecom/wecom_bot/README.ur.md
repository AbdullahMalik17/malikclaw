# کاروباری وی چیٹ (WeCom) روبوٹ

کاروباری وی چیٹ روبوٹ ایک تیز رفتار رسائی کا طریقہ ہے جو کاروباری وی چیٹ (WeCom) کی طرف سے فراہم کیا جاتا ہے، جس کے ذریعے Webhook URL سے پیغامات وصول کیے جا سکتے ہیں۔

## کنفیگریشن

```json
{
  "channels": {
    "wecom": {
      "enabled": true,
      "token": "YOUR_TOKEN",
      "encoding_aes_key": "YOUR_ENCODING_AES_KEY",
      "webhook_url": "https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=YOUR_KEY",
      "webhook_path": "/webhook/wecom",
      "allow_from": [],
      "reply_timeout": 5
    }
  }
}
```

| فیلڈ             | قسم   | لازمی | تفصیل                                         |
| ---------------- | ------ | ---- | -------------------------------------------- |
| token            | string | ہاں   | دستخط کی تصدیق کا ٹوکن                                 |
| encoding_aes_key | string | ہاں   | ڈی کرپشن کے لیے 43 حروف پر مشتمل AES کلید                  |
| webhook_url      | string | ہاں   | کاروباری وی چیٹ گروپ چیٹ روبوٹ کا Webhook URL جہاں جواب بھیجے جائیں گے |
| webhook_path     | string | نہیں   | Webhook اینڈ پوائنٹ کا راستہ (ڈیفالٹ: /webhook/wecom)     |
| allow_from       | array  | نہیں   | صارف ID کی سفید فہرست (خالی قدر = تمام صارفین کی اجازت ہے)        |
| reply_timeout    | int    | نہیں   | جواب کا ٹائم آؤٹ (سیکنڈز میں، ڈیفالٹ قدر: 5)          |

## سیٹ اپ کا طریقہ کار

1. کاروباری وی چیٹ گروپ میں روبوٹ شامل کریں۔
2. Webhook URL حاصل کریں۔
3. (اگر پیغامات وصول کرنے ہوں) روبوٹ کے کنفیگریشن پیج پر پیغامات وصول کرنے کے لیے API ایڈریس (کال بیک ایڈریس) کے ساتھ ساتھ Token اور EncodingAESKey سیٹ کریں۔
4. متعلقہ معلومات کنفیگریشن فائل میں درج کریں۔

   نوٹ: MalikClaw اب تمام چینلز کے ویب ہک کال بیکس وصول کرنے کے لیے ایک مشترکہ Gateway HTTP سرور استعمال کرتا ہے، جو ڈیفالٹ طور پر 127.0.0.1:18790 پر سنتا ہے۔ اگر آپ پبلک نیٹ ورک سے کال بیکس وصول کرنا چاہتے ہیں، تو براہ کرم بیرونی ڈومین کو گیٹ وے (ڈیفالٹ پورٹ 18790) پر ریورس پراکسی کریں۔
