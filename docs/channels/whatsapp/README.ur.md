# واٹس ایپ چینل کنفیگریشن گائیڈ 📱

ملک کلاؤ (MalikClaw) واٹس ایپ کو دو اہم طریقوں سے سپورٹ کرتا ہے: **نیٹو (Native)** اور **برج (Bridge)**۔

---

## 1. نیٹو واٹس ایپ (Native WhatsApp - تجویز کردہ)

نیٹو موڈ براہ راست ملک کلاؤ کے اندر [whatsmeow](https://github.com/tulir/whatsmeow) لائبریری کے ذریعے کام کرتا ہے۔ یہ سب سے آسان طریقہ ہے لیکن اس کے لیے ایک مخصوص بلڈ ٹیگ کی ضرورت ہوتی ہے۔

### نیٹو سپورٹ کے ساتھ بلڈ کریں
نیٹو واٹس ایپ استعمال کرنے کے لیے، آپ کو ملک کلاؤ کو `whatsapp_native` ٹیگ کے ساتھ بلڈ کرنا ہوگا:
```bash
make build-whatsapp-native
# یا
go build -tags whatsapp_native ./cmd/malikclaw/...
```

### کنفیگریشن (Configuration)
اپنی `config.json` فائل کو اپ ڈیٹ کریں:
```json
{
  "channels": {
    "whatsapp": {
      "enabled": true,
      "use_native": true,
      "session_store_path": "",
      "allow_from": []
    }
  }
}
```
- `use_native`: اسے `true` پر سیٹ کریں۔
- `session_store_path`: اگر خالی چھوڑ دیا جائے تو یہ خود بخود `<workspace>/whatsapp/` میں محفوظ ہوگا۔
- `allow_from`: واٹس ایپ آئی ڈیز (جیسے `923XXXXXXXXX@s.whatsapp.net`) شامل کریں تاکہ صرف مخصوص لوگ اسے استعمال کر سکیں۔

### پہلی بار رن اور پیئرنگ (Pairing)
1. ٹرمینل میں `malikclaw gateway` رن کریں۔
2. ٹرمینل میں ایک **QR کوڈ** نظر آئے گا۔
3. اپنے فون پر واٹس ایپ کھولیں -> **Linked Devices** -> **Link a Device**۔
4. ٹرمینل میں موجود QR کوڈ کو اسکین کریں۔
5. آپ کا سیشن مستقبل کے لیے محفوظ ہو جائے گا۔

## 2. واٹس ایپ برج (WhatsApp Bridge)

برج موڈ آپ کو ملک کلاؤ کو ایک بیرونی واٹس ایپ ویب ساکٹ برج سے جوڑنے کی اجازت دیتا ہے۔ یہ انوائرمنٹ کے لیے مفید ہے جہاں آپ ملک کلاؤ کی بائنری کو چھوٹا رکھنا چاہتے ہیں۔

### کنفیگریشن (Configuration)
اپنی `config.json` فائل کو اپ ڈیٹ کریں:
```json
{
  "channels": {
    "whatsapp": {
      "enabled": true,
      "use_native": false,
      "bridge_url": "ws://your-bridge-url:3001",
      "allow_from": []
    }
  }
}
```
- `use_native`: اسے `false` پر سیٹ کریں۔
- `bridge_url`: اپنے واٹس ایپ برج کا ویب ساکٹ URL درج کریں۔

## 💬 استعمال (Usage)

ایک بار جڑ جانے کے بعد، آپ ملک کلاؤ سے کسی بھی دوسرے واٹس ایپ کانٹیکٹ کی طرح بات کر سکتے ہیں۔
- **پرائیویٹ چیٹ**: ملک کلاؤ براہ راست جواب دے گا۔
- **گروپس**: `@malikclaw` مینشن یا پریفکس (اگر کنفیگر کیا گیا ہو) استعمال کریں۔

## ⚠️ حفاظتی نوٹس (Security Notice)

- واٹس ایپ سیشنز آپ کے ورک اسپیس میں SQLite ڈیٹا بیس کے طور پر محفوظ ہوتے ہیں۔ ان فائلوں کو محفوظ رکھیں۔
- `allow_from` کا استعمال کریں تاکہ صرف مجاز صارفین ہی آپ کے اے آئی اسسٹنٹ سے بات کر سکیں۔
